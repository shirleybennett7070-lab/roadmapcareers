# Additional Job APIs You Can Add

## Free & Working Options

### 1. **Jooble API** (FREE - Recommended!)
- **Cost:** Free tier available
- **Coverage:** International, including US
- **Quality:** Good, aggregates from multiple sources
- **Setup:** https://jooble.org/api/about
  - Simple registration
  - Get API key instantly
  - 500 requests/day free

### 2. **Findwork.dev** (FREE)
- **Cost:** Free
- **Coverage:** Remote jobs, international
- **Quality:** Decent, focuses on remote work
- **Setup:** https://findwork.dev/developers/
  - No API key required for basic use
  - Rate limited but generous

### 3. **Reed.co.uk API** (FREE)
- **Cost:** Free
- **Coverage:** Mostly UK, some international
- **Quality:** Excellent descriptions
- **Setup:** https://www.reed.co.uk/developers
  - Register for API key
  - Free tier: 500 calls/month

### 4. **Careerjet API** (FREE)
- **Cost:** Free with attribution
- **Coverage:** Global, 90+ countries
- **Quality:** Good aggregator
- **Setup:** http://www.careerjet.com/partners/api/
  - Request API key (approved within hours)
  - Must display "Jobs by Careerjet" attribution

### 5. **Kombo Jobs API** (FREE tier)
- **Cost:** Free tier available
- **Coverage:** Multiple ATS systems
- **Quality:** Very good
- **Setup:** https://kombo.dev/
  - More for enterprise, but has free tier
