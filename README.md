# RoadmapCareers

Full-stack job lead generation and training platform.

## Project Structure

- `backend/` - Node.js API and services
- `frontend/` - React UI (coming soon)
- `shared/` - Shared utilities

## Getting Started

See [backend/README.md](./backend/README.md) for setup instructions.

## Modules

- **Jobs** - Fetch and store remote jobs in Google Sheets
- **Email** - Auto-reply system (in progress)
- **Quiz** - User onboarding (planned)
